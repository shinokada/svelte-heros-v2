# Svelte Heros v2

A collection of reusable Svelte components for building user interfaces for Svelte Runes web applications.

- CodeWrapper
- Footer
- helpers
- HighlightCompo
- HomeCards
- IconPage
- SupportBanner
- TechInfo


## Requirements

- Svelte 5 (Runes)
- SvelteKit

## Installation

```sh
pnpm i -D runes-webkit svelte-rune-highlight highlight.js
```
